# Classification of High-Dimensional Data

This repository contains my Final Year Project completed as part of the Bachelor of Science (Honours) in Mathematics and Statistics at Hong Kong Baptist University.

## Project Objective

The project investigates how different machine learning classification algorithms perform on high-dimensional datasets. Specifically, it compares:

* Support Vector Machine (SVM)
* Random Forest (RF)
* Neural Network (NN)

using both synthetic and real-world datasets.

## Key Skills Demonstrated

* Machine Learning
* Classification Modelling
* Data Preprocessing
* Feature Engineering
* Dimensionality Reduction (PCA)
* TF-IDF Vectorisation
* Cross Validation
* Python
* Scikit-Learn
* Data Visualisation
* Statistical Analysis

## Datasets

* Breast Cancer Dataset
* Adult Income Dataset
* MNIST Handwritten Digits Dataset
* Synthetic High-Dimensional Dataset
* Genomic-Style High-Dimensional Dataset
* Text Classification Dataset

## Repository Structure

The repository contains two implementations:

### original_style_reconstruction.py

A reconstruction of the workflow most closely matching the original project development process.

### extended_thesis_reconstruction.py

A comprehensive reconstruction implementing all methodologies described in the final dissertation.

## Results

The project demonstrates that no single classification algorithm is universally optimal. Performance depends heavily on data characteristics such as dimensionality, sample size, feature redundancy, and non-linearity.
